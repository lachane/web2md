1. 多平台触发方式
   - Windows: 右键菜单「保存为 Markdown」
   - macOS: 
     * Control + 点击显示上下文菜单
     * 双指轻触触控板显示菜单
   - 通用方式：
     * 工具栏图标点击（显示简单下拉菜单）
     * 添加基础快捷键 (Command/Ctrl + Shift + S)

2. 保存选项
   - 右键菜单显示子选项：
     * 新建文档保存
     * 附加到现有文档
   - 工具栏下拉菜单：
     * 显示保存模式选项
     * 记住上次选择

3. 状态提示
   - 转换成功提示
   - 保存成功提示
   - 附加成功提示
   - 基础错误提示
