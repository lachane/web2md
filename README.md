🔧 安装未打包 Chrome 插件的方法
1. 下载源码
   一般是一个 .zip 压缩包，里面包含 manifest.json 文件和插件代码。
   先解压到一个你容易找到的文件夹里，比如桌面。
   打开 Chrome 扩展管理页
   在 Chrome 地址栏输入：
   chrome://extensions/
   或者：点击右上角菜单 → 更多工具 → 扩展程序。

2. 开启开发者模式
   在扩展管理页右上角，打开 “开发者模式” 开关。
   加载已解压的扩展程序
   点击左上角的 “加载已解压的扩展程序” 按钮。
   选择你刚才解压的插件文件夹。

3. 完成安装 🎉
   加载成功后，你会看到插件出现在扩展程序列表里。
   插件有按钮，可以在浏览器右上角扩展区看到。
